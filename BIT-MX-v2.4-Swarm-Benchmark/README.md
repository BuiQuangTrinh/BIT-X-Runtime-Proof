## Overview

BIT-MX v2.4 is an engineering research prototype exploring whether a supervisory boundary-aware runtime layer can improve swarm topology survival under extreme disturbances.

The framework does not replace:
- Newtonian mechanics
- PID controllers
- classical swarm algorithms

Instead, BIT-MX acts as a:
- supervisory runtime layer
- topology survival monitor
- boundary-aware meta-controller
